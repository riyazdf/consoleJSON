Documentation

consoleJSON.log(theJSON, [, ruleset])

It's as simple as that, really. The second parameter is optional. If not provided, this will use the default setup and themes, logging the JSON with bolded keys and newline delimited key-value pairs. If a second parameter is provided, consoleJSON will use this as the ruleset when displaying the JSON.

consoleJSON.filter(theJSON, key)

You can also filter the JSON by keys.

consoleJSON.Ruleset([Theme])

There are several built-in themes that configure the color of JSON primitives. Beyond that, rules can be defined and added to a ruleset.

consoleJSON.Rule(type, val, attr, target)

These are the building blocks behind Rulesets. The user can specify CSS-like attributes and values for certain keys.
