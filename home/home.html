Simple


Command line JSON formatting is ugly, and makes for tedious debugging. With ConsoleJSON, you can see JSON with the tools you know in the development environment you're familiar with.

Elegant


ConsoleJSON comes with default settings to indent nested groups, and to colorize keys. You can always change these settings or configure your own with our built in functionality.

Setup

Setup is incredibly easy. Simply download the consoleJSON.js file above, and include it in your project structure. Then, include the following code in your html document:

<script src="path/to/consoleJSON.js"></script>
